Vibe Check: Evaluating LLM-Generated Code

DO NOT RUN ANY FILES THAT AREN'T IN THE 'Pipeline' FOLDER ON YOUR MACHINE!!

Overview

This project presents a reproducible pipeline for evaluating the security and maintainability of code generated by Large Language Models (LLMs).

The system automatically:

Sends a prompt to an LLM
Saves the generated code
Runs static analysis using SonarQube
Extracts findings into structured CSV datasets

The goal is to create a consistent, comparable evaluation framework for analysing how different LLMs perform when generating code.

Key Features
Automated LLM → analysis pipeline
Per-run code archival for reproducibility
Static analysis using SonarQube
Structured export of issues to CSV
Designed for multi-model comparison (Gemini, ChatGPT, Claude)

How It Works
1. Code Generation

The pipeline reads a prompt and sends it to an LLM (currently Gemini).

2. Archiving

Each output is saved with a timestamp:

Outputs/Gemini/YYYY-MM-DD_HHMMSS_app.py

(DO NOT EXECUTE THESE FILES ON YOUR MACHINE!)

This ensures reproducibility, traceability, and comparison across runs and models.

3. Static Analysis

SonarQube scans the generated file and detects:

bugs
code smells
maintainability issues
limited security concerns

4. Data Extraction
Results are exported to:

Scoring/sonar_issues.csv

Each row represents a detected issue.

2026-03-23 12:53:31,gemini,Outputs/Gemini/...,CODE_SMELL,MINOR,python:S1481,Remove unused variable,...

Setup

1. Install dependencies

pip install -r requirements.txt

2. Configure environment variables

Create a .env file:

GEMINI_API_KEY=your_key_here
SONAR_LOGIN=your_sonar_token

3. Start SonarQube

Ensure SonarQube is running locally:

http://localhost:9000

4. Run the pipeline

python Pipeline/run_pipeline.py

Notes on Security

This repository intentionally contains insecure or low-quality generated code.

These files are used as evaluation inputs, benchmarking artefacts, and test cases for static analysis tools. They are not intended for production use.
DO NOT EXECUTE THESE FILES ON YOUR MACHINE!

Limitations

SonarQube provides limited security detection for Python
Some vulnerabilities may not be flagged
Results depend on prompt design and model behaviour
